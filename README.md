# CleanCityPortal 

## 📌 Project Overview

*CleanCityPortal is a client-side web application where users can report not only road breakages but also any kind of city-related problems. Users can visit the website to add issues with relevant details and images, helping authorities and communities identify and address problems efficiently.

---

## 🎯 Purpose

The main goal of this project is to provide a **simple and user-friendly platform** where users can easily report road problems with images, helping authorities identify and resolve issues faster while maintaining a clean and safe city environment.

## ✨ Core Features

- 📸 Upload images of  issues (e.g., road breakage)
- ➕ Add new issues reports
- 🗑️ Delete reported issues when resolved or unnecessary
- 📋 View a list of all reported issues
- 📱 Fully responsive and user-friendly UI
- ⚡ Fast performance with modern frontend tools
